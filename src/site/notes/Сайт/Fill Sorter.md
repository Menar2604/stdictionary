## **Сортировщик по заполненности**
`IsEmpty`, `IsFull`

#StorageTechContraption 

Сортировщик по заполненности - это [[Сайт/Box Separator\|Разделитель шалкеров]], который имеет разные выходы в зависимости от того, насколько заполнен шалкер, обычно пустой / полный / частично полный. 
==isEmpty== - это сортировщик, который имеет выход только для пустых и непустых ( полных / частично полных ) 
==isFull== - это сортировщик, который имеет выход только для полных и неполных (пустых / частичных )
