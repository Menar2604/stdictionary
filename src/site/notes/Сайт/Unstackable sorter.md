## **Сортировщик нестакаемых предметов**

#StorageTechContraption 

Сортировщик предметов, который сортирует нестакаемые предметы по различным категориям. Обычные сортировщики предметов нельзя использовать в этом случае, поскольку предметы не складываются в стопку, поэтому для их разделения необходимо использовать уникальные свойства каждого типа предметов. Не путать с [[Сайт/Unstackable filter\|Фильтром нестакаемых предметов]].
