## **Детектор обновления жидкости**
ДОЖ

#Contraption 

Детектор обновления жидкости (ДОЖ) - это устройство, которое использует жидкости для обнаружения специального набора обновлений, некоторые из которых могут быть обнаружены с помощью [[Сайт/Block Update Detector\|Детектора Обновления Блока]], [[Сайт/Comparator Update Detector\|Детектора обновления компаратора]] или с помощью наблюдателя, но некоторые из них обнаруживаются исключительно с помощью ДОЖей. После обнаружения обновления и запуска он обычно сбрасывает жидкость в состояние, при котором он сможет снова обнаружить следующее обновление.
