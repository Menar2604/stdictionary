## **Мультисортировщик**
MIS

#StorageTechContraption 

Механизм, который сортирует несколько типов предметов в один сундук или группу сундуков, обычно используется для экономии пространства, используемого сортировщиками отдельных типов предметов для группировки не слишком распространенных предметов по категориям внутри системы склада.
