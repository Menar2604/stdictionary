## **Точная воронка**

#Feature #StorageTechContraption 

Воронка, которая разблокируется в тот же момент, когда она должна засасывать предмет. Если объект предмета создан в этот игровой такт, как это было бы, например, при правильном определении времени ломании шалкера, для этого предмета будет применено 0 физических параметров. Это делает его не только чрезвычайно надёжным, поскольку в нём не будет проблем с физикой, но также может быть совместимо с использованием ленивых чанков.
